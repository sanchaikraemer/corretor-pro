# GRAFO DE AGENTES - RESUMO

Fluxo oficial:

Corretor / Usuario
-> Frontend PWA Android/iOS/Desktop
-> Direciona Core
-> Agente ZIP
-> Agente Transcricao
-> Agente Timeline
-> Agentes Perfil, Objecoes, Timing e Memoria
-> Agentes Cerebro, Similaridade e Aprendizado
-> Agente Respostas
-> Agente Material Inteligente
-> Agente Agenda
-> Agente Pipeline
-> Agente Dashboard
-> Interface do Corretor
-> Acoes praticas
-> Aprendizado

A interface final deve mostrar apenas:
- Quem atender
- Por que atender
- Quando retomar
- Qual acao tomar
- O que falar

Os arquivos HTML de grafo foram colocados na pasta 08_REFERENCIAS_VISUAIS_GRAFOS.
