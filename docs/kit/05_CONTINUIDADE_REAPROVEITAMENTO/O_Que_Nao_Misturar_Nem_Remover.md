# O QUE NAO MISTURAR NEM REMOVER

## Nao misturar
- Projeto antigo inteiro com projeto atual.
- Visual antigo com visual oficial atual.
- Codigo antigo sem entender a funcao.
- Prompt de continuidade com manual comercial.
- Historico de erro com documento de apresentacao.
- Comandos tecnicos com DNA do produto.

## Nao remover sem autorizacao
- Importacao de conversa.
- Processamento de ZIP.
- Transcricao de audios.
- Timeline.
- Analise comercial.
- Respostas sugeridas.
- Lead inteligente.
- Pipeline.
- Agenda.
- Vendas.
- Cerebro Comercial.
- Memoria do lead.
- Material inteligente.

## Nao aceitar como pronto
- Apenas build aprovado.
- Apenas tela bonita.
- Apenas cadastro de lead.
- Apenas gerador de mensagem.
- Sistema que nao processa conversa real.
- Sistema que ignora audio.
- Sistema que nao mostra proxima acao.
