# CHECKLIST MVP OBRIGATORIO

Uma versao do Direciona so pode ser considerada util se cumprir, no minimo:

## Entrada e processamento
[ ] Importa ZIP real do WhatsApp.
[ ] Funciona no Android.
[ ] Funciona no iOS com alternativa manual.
[ ] Funciona no desktop.
[ ] Le o TXT da conversa.
[ ] Identifica audios.
[ ] Transcreve audios.
[ ] Ignora imagens, videos, emojis, figurinhas e documentos na analise inicial.
[ ] Monta timeline cronologica.
[ ] Intercala texto e audio transcrito na ordem correta.

## Analise comercial
[ ] Analisa a conversa inteira, nao frases soltas.
[ ] Identifica perfil provavel.
[ ] Identifica produto de interesse.
[ ] Detecta objecoes explicitas.
[ ] Detecta objecoes implicitas.
[ ] Calcula dias sem resposta.
[ ] Identifica tipo de retomada.
[ ] Sugere melhor horario.
[ ] Define prioridade.
[ ] Gera proxima acao.

## Respostas e acao
[ ] Gera exatamente 3 mensagens: Direta, Consultiva e Retomada.
[ ] Mensagens sao naturais e prontas para WhatsApp.
[ ] Mensagens usam blocos curtos.
[ ] Mensagens nao usam retomada generica.
[ ] Ha botao copiar.
[ ] Ha botao abrir WhatsApp.
[ ] Ha acao para marcar retorno.

## Organizacao
[ ] Cria ou atualiza lead real.
[ ] Salva historico.
[ ] Atualiza pipeline.
[ ] Organiza agenda.
[ ] Mostra fila diaria.
[ ] Mostra lead prioritario.
[ ] Oculta vendido/perdido/standby da pipeline principal quando aplicavel.

## Experiencia
[ ] Home mostra quem atender, por que, quando e o que falar.
[ ] Nao mostra erro tecnico cru.
[ ] Processamento tem barra/etapas/percentual.
[ ] Interface e mobile-first.
[ ] Visual preto/grafite com neon pontual.
[ ] Nao parece CRM pesado.

Se algum item central de ZIP, audio, timeline, analise e mensagem falhar, o produto ainda nao esta pronto.
